# HDRezka Custom
Минимальное легковесное приложение для просмотра контента с сервиса HDRezka

> [!IMPORTANT]
> Данное приложение содержит лишь базовый функционал и не является полноценной заменой других приложений. Делал исключительно под себя и под слабенький проектор.

#### Функциионал:
* Поиск контента
* Просмотр с выбором озвучки\сезона\эпизода\качества
* Список просмотренного и возможность продолжить просмотр (если введены данные для входа в учетную запись hdrezka)
